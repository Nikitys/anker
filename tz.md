## 1. Общая концепция сайта

### 1.1 Позиционирование
Сайт продаёт **анкерные системы для геотехники** в РФ. Главное УТП — **не просто поставка оборудования, а полный цикл: проектирование + поставка + авторский надзор**. Проектирование является и отдельной услугой, и аргументом доверия («мы сами рассчитываем — мы отвечаем за результат»).

### 1.2 Тон и стиль текстов
- Профессиональный, технически грамотный, без лишней «маркетинговой воды»
- Апелляция к нормативной базе РФ (ГОСТ, СП, СНиП)
- Конкретные цифры: несущие нагрузки, глубины, сроки, гарантии
- Нет общих фраз типа «мы лучшие» — только факты и аргументы

### 9.3 Анимации — перечень и поведение

#### Hero-блок
- **Parallax** на фоновом изображении при скролле (смещение 20–30% от скорости страницы)
- **Fade-in + slide-up** для заголовка и подзаголовка при загрузке (задержка 0.3с / 0.5с)
- **Пульсация** на основной CTA-кнопке (scale 1→1.03→1, бесконечно, ease-in-out, период 2.5с)

#### Блок УТП (4 плитки)
- **Staggered появление** карточек при входе в viewport: каждая с задержкой +0.15с (Intersection Observer)
- **Hover-эффект** на плитке: лёгкое поднятие (translateY -4px) + усиление тени

#### Счётчики (реализованные объекты, метры анкеров, кН нагрузки)
- **Анимированный countUp** при попадании блока в viewport: от 0 до целевого значения за 1.8с, easing ease-out
- Пример: «0 → 347 объектов», «0 → 2 400 000 кН»

#### Каталог / карточки товаров
- **Hover на карточке**: плавное появление оверлея с кнопками «Подробнее» / «Запросить КП» (opacity 0→1, 0.25с)
- **Фильтры**: при переключении фильтра карточки исчезают/появляются с fade (opacity + scale, 0.3с)

#### Страница товара
- **Sticky-шапка характеристик**: при скролле вниз таблица характеристик «прилипает» к верху экрана
- **Схема анкера (SVG)**: при загрузке страницы линии схемы «прорисовываются» через stroke-dashoffset анимацию (1.2с)

#### Страница «Проектирование» — таймлайн шагов
- **Прогрессивное появление** шагов при скролле: каждый шаг появляется когда входит в viewport
- Соединительная линия между шагами «дорисовывается» сверху вниз (height 0→100%, 0.4с per step)

#### Форма расчёта
- **Плавное раскрытие** дополнительных полей при выборе типа объекта (max-height transition)
- **Shake-анимация** на незаполненных обязательных полях при попытке отправить

#### Общие правила анимаций
- Все анимации только через CSS transitions / CSS animations или GSAP — никакого jQuery
- `prefers-reduced-motion`: все анимации отключаются если пользователь запросил минимум движения
- Длительность: не более 0.5с для hover-эффектов, не более 1.5с для появления элементов
- Не анимировать одновременно более 3 элементов на экране
